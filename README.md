# Homework 1 Coding Assistant Onboarding

This repository contains my work for Homework 1 in the Generative AI course. It shows how I used a coding assistant, made my own manual edits, and tracked the process with Git and GitHub.

## Repository Contents

- `assistant_draft.md`: the first draft created with the coding assistant
- `final_version.md`: my manually edited final version
- `README.md`: a summary of the workflow and Git steps used

## Git Steps I Used

I created a public GitHub repository, cloned it to my local computer, and created the required files. Then I used `git add`, `git commit`, and `git push` to save and upload each step of the assignment.
